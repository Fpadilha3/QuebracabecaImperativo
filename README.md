# 🧩 Quebra-Cabeça Interativo

Projeto desenvolvido na disciplina de Programação Web I.

Este projeto tem como objetivo praticar **HTML, CSS e JavaScript**, explorando conceitos de interação com o usuário e manipulação do DOM.

---

## 🚀 Acesse o projeto

🔗 Repositório:  
https://github.com/Fpadilha3/QuebracabecaImperativo

---

## 🎯 Objetivo

O sistema simula um **quebra-cabeça interativo**, onde o usuário pode ativar uma peça e visualizar mudanças dinâmicas na interface.

---

## 🧠 Conceitos aplicados

- Estruturação com HTML
- Estilização com CSS moderno
- Manipulação do DOM
- Eventos com JavaScript
- Alteração dinâmica de conteúdo
- Uso de classes CSS para estados (ativo / bloqueado)

---

## 🧩 Funcionalidades

- Ativar e desativar a peça interativa
- Mudança de estado visual (bloqueado → ativo)
- Atualização de texto e ícones dinamicamente
- Interface responsiva e estilizada
